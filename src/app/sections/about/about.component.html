El Sistema de Transición se concibe como un Ecosistema de Solución que permite gestionar la información necesaria para realizar el barrido predial por parte de las entidades y actores involucrados en el proceso del barrido predial masivo de manera transitoria mientras los sistemas misionales están en proceso de ajustes para responder a las nuevas necesidades. En este sentido el ST busca apoyar a los gestores catastrales en el flujo de información con los operadores y demás actores.

Herramientas: El Sistema de Transición se articula con el Asistente LADM-COL, esta herramienta interactúa con esté sistema en tareas que requieren análisis y validación de datos espaciales.
